#Prismagram2_Backend

Clone Instagram Backend with GraphQL + Prisma + mongoDB

## User Stories

- [x] Create Account
- [x] Request Secret
- [x] Confirm Secret (Log In)
- [ ] See the Feed
- [x] Like / Unlike a Post
- [ ] Comment on a Post
- [ ] Upload a Post
- [ ] Edit a Post (Delete)
- [ ] See full Post
- [ ] See User Profile
- [ ] Edit Profile
- [ ] Follow / Unfollow
- [ ] Search by User
- [ ] Search by location
